# Personal Portfolio

A responsive personal portfolio website built with React, Vite, Tailwind CSS, and React Router.

## Tech

- React
- Vite
- Tailwind CSS
- React Router
- Tailwind CSS

## Getting started

**Requirements:** Node.js 18+

```bash
npm install
npm run dev
```

Open: http://localhost:5173

## Scripts

```bash
npm run dev       # start dev server
npm run build     # production build
npm run preview   # preview production build
npm run lint      # run ESLint
```

## Pages / Routes

- `/` — Home
- `/about` — About
- `/contact` — Contact
- `/profile` — Profile
- `/project` — Projects

## Project structure

```txt
src/
  main.jsx
  App.jsx
  Pages/
    Home.jsx
    About.jsx
    Contact.jsx
    Profile.jsx
    Project.jsx
```

## Customize

Edit the page components in `src/Pages/` and update navigation/routes in `src/App.jsx`.

## Deploy

```bash
npm run build
```

Deploy the `dist/` folder (Vercel, Netlify, GitHub Pages, etc.).
